# InBody Progress Tracker

Веб-приложение для отслеживания динамики состава тела по данным анализатора **InBody**.  
Загружайте PDF-отчёты и наблюдайте за изменениями веса, мышечной массы, процента жира, висцерального жира и основного обмена на удобных графиках.

## Возможности
- Парсинг PDF-файлов InBody (извлекаются все ключевые метрики).
- Хранение измерений в SQLite.
- Интерактивные графики с временной шкалой (Chart.js).
- Адаптивный интерфейс (работает на телефонах).
- Автоматическая установка через `install.sh` с настройкой systemd-сервиса.

## Быстрая установка (на Ubuntu)

```bash
git clone https://github.com/rusak063-arch/inbody-app.git
cd inbody-app
chmod +x install.sh
./install.sh
